\# Military Object Detection System (YOLO)



A complete computer vision pipeline built for military object detection using YOLO.

This project demonstrates dataset handling, model training, inference, GUI deployment,

and executable packaging.



\## Features

\- YOLO-based object detection

\- Drag \& drop + file picker GUI

\- High-contrast bounding boxes

\- Confidence score overlays

\- Status bar with detection summary

\- YOLO-format output (.txt)

\- Windows executable deployment



\## Pipeline

1\. Dataset preprocessing

2\. YOLO training \& fine-tuning

3\. Validation and test inference

4\. GUI-based image detection

5\. Executable packaging
